Historical Volatility Tracker

Paste a price series (or simulate one) and compare realized-volatility estimators: close-to-close, EWMA (RiskMetrics) and rolling-window dynamics.

Input Prices

One closing price per line (or comma-separated). Daily data assumed; annualization uses 252 days.

Results

Blue: rolling close-to-close annualized vol. Orange: EWMA vol. The EWMA reacts faster to regime changes but never fully forgets — exactly the trade-off GARCH formalizes.

Estimator Notes

  • Close-to-close: $\hat\sigma^2 = \frac{252}{n}\sum r_i^2$ (zero-mean convention — for daily data, estimating the mean adds more noise than it removes).
  • EWMA (RiskMetrics): $\sigma_t^2 = (1-\lambda) r_{t-1}^2 + \lambda\sigma_{t-1}^2$, λ = 0.94 daily — an integrated GARCH with effective memory $\approx 1/(1-\lambda) \approx 17$ days.
  • Range estimators (Parkinson, Garman-Klass, Yang-Zhang) use OHLC data for ~5× efficiency — preferable when highs/lows are available.
  • Window choice is a bias-variance dial: 21d tracks regimes but is noisy (s.e. ≈ $\sigma/\sqrt{2n}$ ≈ 15% relative); 252d is stable but stale.
