Non-Homogeneous Case - Method of Undetermined Coefficients

Guided guesswork methods for finding particular integrals, exponential/polynomial/trigonometric RHS, and failure case overrides.

1. Particular Solutions & Linearity

For a non-homogeneous differential equation $Ly = g(x)$, the complete general solution is composed of two distinct components:

$$y = y_c + y_p$$

where $y_c$ solves the homogeneous equation $Ly = 0$, and $y_p$ is a particular solution satisfying $Ly_p = g(x)$. Substituting $y_c + y_p$ into the operator confirms linearity: $L(y_c + y_p) = L(y_c) + L(y_p) = 0 + g(x) = g(x)$.

2. Standard RHS Trial Functions

The Method of Undetermined Coefficients is a guided guesswork approach where the form of the trial solution $y_p$ is determined by the mathematical category of the right-hand side function $g(x)$:

Category A: Exponential RHS $g(x) = M e^{kx}$

Trial Solution: $y_p = C e^{kx}$ (provided $k$ is not a root of the auxiliary equation).

Example: For $y'' - 5y' + 6y = 3e^{5x}$, try $y_p = Ce^{5x}$. Subbing in yields $C = 1/14 \implies y_p = (1/14)e^{5x}$.

Category B: Polynomial RHS $g(x) = P_n(x)$

Trial Solution: A general polynomial of the same degree: $y_p = p_0 + p_1 x + \dots + p_n x^n$.

Example: For $y'' - 5y' + 6y = x^2$, try $y_p = p_0 + p_1 x + p_2 x^2$. Substituting and equating coefficients yields $p_2 = 1/2$, $p_1 = -3/2$, and $p_0 = 7/4$.

Category C: Trigonometric RHS $g(x) = M \cos\omega x$ or $N \sin\omega x$

Trial Solution: Both sine and cosine must always be included together: $y_p = a \cos\omega x + b \sin\omega x$.

Euler's Complex Shortcut: Recognize $\cos\omega x = \text{Re}(e^{i\omega x})$. Solve for the complex particular solution using trial $y_p = K e^{i\omega x}$, then extract the real part. This is mathematically much more efficient.

3. Overcoming the Failure Case

A failure case arises when the exponential parameter $k$ in $g(x)$ is already a root of the auxiliary equation, meaning it appears in the complementary function $y_c$. Substituting the standard trial solution would yield zero on the left-hand side, making it impossible to match the right-hand side.

The Correction Principle:

We scale the standard trial solution by multiplying it by $x$ to achieve linear independence:

$$\text{New Trial } y_p = x \cdot \left( C e^{kx} \right)$$

If $k$ is a double repeated root of the auxiliary equation, we scale the trial solution by multiplying by $x^2$:

$$\text{New Trial } y_p = x^2 \cdot \left( C e^{kx} \right)$$

General Rule: The multiplying power of $x$ corresponds exactly to the multiplicity of the root.

Plain English Notes

The Big Picture: When an equation is driven by an active right-hand side force ($g(x)$), we must find a specific solution ($y_p$) to balance it.

  • Complementary vs. Particular: The general solution is a team effort:
    • The **Complementary Function ($y_c$)** handles the background physics of the system.
    • The **Particular Integral ($y_p$)** forces the system to react to the driving input.
  • RHS Matching Guesswork: We guess $y_p$ to match the shape of the driving force:
    • If it's an exponential $e^{kx}$, guess $C e^{kx}$.
    • If it's a polynomial $x^n$, guess a general polynomial of the same degree.
    • If it's a sine or cosine, guess a mix of sines and cosines ($A \cos x + B \sin x$) because derivatives swap them.
  • The Failure Case Trap: If your guess ($e^{kx}$) is already in the complementary function ($y_c$), plugging it in yields $0$. This makes it impossible to solve. We break this conflict by multiplying our trial guess by $x$ (or $x^2$ for double roots), shifting it into a new independent dimension.
