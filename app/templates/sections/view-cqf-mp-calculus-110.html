1.10 Definite Integrals & Substitution Pitfalls

Integral bounds mapping, dummy variables, substitution theorems, and non-invertible transformation traps.

1.10.1 Definite Integrals & Substitution Rule

The definite integral $\int_a^b f(x)dx$ represents the signed geometric area under $f(x)$ bounded by the interval $[a, b]$. Because the variable of integration is bound by the limits, it acts strictly as a dummy variable:

$$\int_a^b f(x)dx = \int_a^b f(y)dy = \int_a^b f(\theta)d\theta$$

The Substitution Rule: Differentiable variable changes require transforming the integration bounds. If $z = g(x) \implies dz = g'(x)dx$:

$$\int_a^b f(g(x))g'(x)dx = \int_{g(a)}^{g(b)} f(z)dz$$

⚠️ CRITICAL ACADEMIC WARNING: The Non-Invertibility Trap
Substitutions like $z = x^2$ fail on symmetric intervals containing the origin (e.g. $[-1, 1]$) because the derivative $\frac{dz}{dx} = 2x$ vanishes at $x=0$. This breaks the mathematical assumption of bijectivity (one-to-one mapping) for the transformation, collapsing the boundary limits ($g(-1) = g(1) = 1$) to yield a false zero result. Always verify that $g'(x) \ne 0$ on the entire integration range.

Exercise 1: Standard Bounds Substitution

Evaluate $\int_0^{\pi/2} \sin^3 x \cos x \, dx$ using trigonometric substitution.

Step-by-Step Solution:
  1. Set up substitution: Let $z = \sin x \implies dz = \cos x \, dx$.
  2. Transform bounds: $$\text{Lower bound: } x=0 \implies z = \sin(0) = 0$$ $$\text{Upper bound: } x=\pi/2 \implies z = \sin(\pi/2) = 1$$
  3. Integrate: $$\int_0^1 z^3 dz = \left[ \frac{z^4}{4} \right]_0^1 = \frac{1}{4}$$
