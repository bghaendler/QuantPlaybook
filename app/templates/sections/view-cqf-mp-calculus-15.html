1.5 Trigonometric & Hyperbolic Mappings

Circular functions periodicities, exponential-driven hyperbolic functions, identities, and inverse logs.

1.5.1 Trigonometric & Hyperbolic Theory

Trigonometric periodicities describe circular oscillations (period $2\pi$). Key reciprocal forms include $\sec x = 1/\cos x$, $\csc x = 1/\sin x$, and $\cot x = 1/\tan x$. Standard circular identity: $\cos^2 x + \sin^2 x = 1$.

Hyperbolic Functions: Defined via exponentials:

$$\sinh x = \frac{e^x - e^{-x}}{2} \quad , \quad \cosh x = \frac{e^x + e^{-x}}{2} \quad , \quad \tanh x = \frac{\sinh x}{\cosh x}$$

Hyperbolic Identities: $\cosh^2 x - \sinh^2 x = 1$, and $\sinh(x+y) = \sinh x\cosh y + \cosh x\sinh y$.

Inverse Hyperbolic Logarithmic Forms: Derived by solving quadratic systems in $e^y$:

$$\sinh^{-1} x = \ln\left(x + \sqrt{x^2 + 1}\right) \quad , \quad \cosh^{-1} x = \ln\left(x + \sqrt{x^2 - 1}\right)$$ $$\tanh^{-1} x = \frac{1}{2}\ln\left(\frac{1+x}{1-x}\right)$$

Exercise 1: Logarithmic sinh⁻¹(x) Derivation

Derive the logarithmic formula for $y = \sinh^{-1} x \implies y = \ln\left(x + \sqrt{x^2 + 1}\right)$.

Step-by-Step Proof:
  1. Start with the exponential definition of $\sinh$: $$y = \sinh^{-1} x \implies x = \sinh y = \frac{e^y - e^{-y}}{2}$$ $$2x = e^y - e^{-y}$$
  2. Multiply through by $e^y$ to form a quadratic equation: $$2x e^y = (e^y)^2 - 1 \implies (e^y)^2 - 2x(e^y) - 1 = 0$$
  3. Solve for $e^y$ using the quadratic formula: Let $u = e^y \implies u^2 - 2xu - 1 = 0$. $$u = \frac{2x \pm \sqrt{4x^2 - 4(1)(-1)}}{2} = x \pm \sqrt{x^2 + 1}$$
  4. Select the positive branch (since $e^y > 0$): $$e^y = x + \sqrt{x^2 + 1} \implies y = \ln\left(x + \sqrt{x^2 + 1}\right) \quad \blacksquare$$
