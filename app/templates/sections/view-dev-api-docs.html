The FastAPI backend behind this application: endpoints, payload reference, and live testing against the running server.

Endpoints

MethodPathPurpose
GET/ Serves this single-page application (Jinja2-rendered index.html).
GET/api/data Term-structure laboratory data: bootstrapped curve, pseudoinverse smoothing, PCA factors, Lorimier spline fit (Swiss yields).
POST/calculate The option engine. JSON payload selects model + parameters; returns price, Greeks, graph/heatmap arrays, vol surfaces and distribution data.
GET/docs FastAPI's auto-generated OpenAPI/Swagger UI — interactive schema for everything above.

Client Snippets

        
curl

        
curl -s -X POST http://localhost:8000/calculate \
  -H "Content-Type: application/json" \
  -d '{"model":"bsm","type":"call","spot":100,"strike":100,"time":1,"rate":0.05,"volatility":0.2}' | jq .price

        
Python (requests)

        
import requests

payload = {"model": "baw", "type": "put", "spot": 100, "strike": 110,
           "time": 0.5, "rate": 0.05, "volatility": 0.25, "dividend": 0.03}
r = requests.post("http://localhost:8000/calculate", json=payload, timeout=10)
r.raise_for_status()
data = r.json()
print(data["price"], data.get("delta"), data.get("gamma"))
