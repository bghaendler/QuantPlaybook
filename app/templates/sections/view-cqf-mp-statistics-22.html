2.2 Maximum Likelihood Estimation (MLE)

Concept and definition of likelihood/log-likelihood, parameter fitting, and Normal distribution derivations.

2.2.1 The Concept of Likelihood

The Maximum Likelihood Estimation (MLE) is a statistical method used for fitting data to a model (Data analysis).

We are asking the question: "Given the set of data, what model parameters are most likely to give this data?"

Note: When using the MLE model we must first assume a distribution, i.e. a parametric model, after which we can try to determine the model parameters.

2.2.2 Motivating Example: Binomial Distribution Parameter $p_0$

Consider data from a Binomial distribution with random variable $X$ and parameters $n = 10$ and $p = p_0$. The parameter $p_0$ is fixed and unknown to us. That is:

$$f(x; p_0) = P(X = x) = \binom{10}{x} p_0^x (1 - p_0)^{10-x}$$

Now suppose we observe some data $X = 3$. Our goal is to estimate the actual parameter value $p_0$ based on this single data observation.

Thought Experiments:
  • Assume $p_0 = 0.5$: $$f(3; 0.5) = P(X = 3) = \binom{10}{3} (0.5)^3 (0.5)^7 \approx 0.117 \quad (\text{Not very high!})$$
  • Assume $p_0 = 0.4$: $$f(3; 0.4) = P(X = 3) = \binom{10}{3} (0.4)^3 (0.6)^7 \approx 0.215 \quad (\text{Better...})$$
General Formulation & Log Maximization:

Let us define the Likelihood function $\ell(p; 3) = f(3; p) = \binom{10}{3} p^3 (1-p)^7$. We want to find the parameter $p$ that maximizes this probability. Taking the natural logarithm to form the Log-Likelihood function $L(p; 3) = \log \ell(p; 3)$:

$$L(p; 3) = 3 \log p + 7 \log(1 - p) + \log \binom{10}{3}$$

To maximize, we set the derivative to zero:

$$\frac{dL}{dp} = \frac{3}{p} - \frac{7}{1 - p} = 0 \implies 3(1 - p) - 7p = 0 \implies 3 - 10p = 0$$

$$p = \frac{3}{10}$$

Thus, the value of $p_0$ that maximizes the observed data probability is $p = \frac{3}{10}$ (or 0.3).

2.2.3 In General

If we have $n$ pieces of iid data $x_1, x_2, x_3, \dots, x_n$ with probability density (or mass) function $f(x_1, x_2, x_3, \dots, x_n; \theta)$, where $\theta$ are the unknown parameter(s). Then the Maximum likelihood function is defined as:

$$\ell(\theta; x_1, x_2, x_3, \dots, x_n) = f(x_1, x_2, x_3, \dots, x_n; \theta)$$

and the log-likelihood function can be defined as:

$$L(\theta; x_1, x_2, x_3, \dots, x_n) = \log \ell(\theta; x_1, x_2, x_3, \dots, x_n)$$

Where the maximum likelihood estimate of the parameter(s) $\theta_0$ can be obtained by maximising $L(\theta; x_1, x_2, x_3, \dots, x_n)$.

2.2.4 Normal Distribution Derivation

Consider a random variable $X$ such that $X \sim N(\mu, \sigma^2)$ with iid observations $x_1, x_2, \dots, x_n$. The joint density is:

$$\ell(\mu, \sigma; x_1, \dots, x_n) = \prod_{i=1}^n f(x_i; \mu, \sigma) = \prod_{i=1}^{n} \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x_i - \mu)^2}{2\sigma^2}}$$

Step 1: Write the log-likelihood function $$L(\mu, \sigma; x_1, \dots, x_n) = \sum_{i=1}^{n} \log f(x_i; \mu, \sigma)$$ $$= \log \left[ \prod_{i=1}^{n} \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x_i - \mu)^2}{2\sigma^2}} \right]$$ $$= -\frac{n}{2} \log(2\pi) - n \log(\sigma) - \frac{1}{2\sigma^2} \sum_{i=1}^{n} (x_i - \mu)^2$$ Step 2: Differentiate partially and solve

To maximize we differentiate partially with respect to $\mu$ and $\sigma$, set the derivatives to zero and solve. Doing so yields:

$$\mu = \frac{1}{n} \sum_{i=1}^{n} x_i$$

$$\sigma^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \mu)^2$$
