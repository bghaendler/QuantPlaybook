1.2 Mapping Theory, Domains & Inverses

Functional mapping classifications, domain boundaries, and inverse pricing transforms.

1.2.1 Rigorous Mapping Classifications

A mathematical function $f: X \to Y$ is a rule that assigns to each element $x \in X$ (the Domain) exactly one element $y \in Y$ (the Codomain). The set of all actual mapped values $f(X) \subseteq Y$ is called the Range.

  • Injective (One-to-One): $f$ is injective if distinct domain elements map to distinct codomain elements: $$\forall x_1, x_2 \in X, \quad f(x_1) = f(x_2) \implies x_1 = x_2$$ Strictly required for the existence of a unique inverse function $f^{-1}(y)$.
  • Surjective (Onto): $f$ is surjective if every element in the codomain $Y$ is mapped to by at least one element in the domain ($f(X) = Y$).
  • Bijective: Both injective and surjective. A bijective mapping has a perfect, unique one-to-one correspondence, guaranteeing a unique, stable inverse transform $f^{-1}: Y \to X$.

Plain English Notes

The Big Picture: A function is a one-way factory that takes inputs (domain) and maps them to outputs (range). Classifying mappings helps us determine if we can safely reverse the process.

  • Invertibility: A function is injective (one-to-one) if every input goes to a unique output. In finance, this is critical: if a pricing model isn't injective, you can't uniquely solve for inputs like implied volatility from a given option price.
  • Venn Correspondence: If a function maps multiple inputs to the same output (like Straddle payoffs mapping $S_1, S_2$ to the same value), we must divide the domain into sub-branches to invert it.

Worked Solutions & Proofs

Exercise 1: Rigorous Injectivity Proof for $f(x) = 3x + 1$
  1. State the Injectivity Condition: To prove injectivity, we assume $f(x_1) = f(x_2)$ and show that it must logically imply $x_1 = x_2$.
  2. Substitute the algebraic rule: $$f(x_1) = f(x_2) \implies 3x_1 + 1 = 3x_2 + 1$$
  3. Subtract 1 from both sides: $$3x_1 = 3x_2$$
  4. Divide by 3: $$x_1 = x_2$$ Since $f(x_1) = f(x_2) \implies x_1 = x_2$, the function is rigorously proven to be injective. $\blacksquare$
Exercise 2: Deriving Option Straddle Non-Injectivity
  1. Let strike price $K = 100$.
  2. Choose two distinct domain elements: $S_1 = 105$ and $S_2 = 95$ ($S_1 \neq S_2$).
  3. Evaluate payoffs: $$V(S_1) = |105 - 100| = |5| = 5$$ $$V(S_2) = |95 - 100| = |-5| = 5$$
  4. Since $S_1 \neq S_2$ but $V(S_1) = V(S_2)$, the payoff is many-to-one, confirming it is non-injective and therefore has no unique inverse function without branch partition.
