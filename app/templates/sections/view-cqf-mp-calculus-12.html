1.2 Mapping Theory, Domains & Inverses

Functional mapping classifications, domain boundaries, and inverse pricing transforms.

1.2.1 Rigorous Mapping Classifications

A mathematical function $f: X \to Y$ is a rule that assigns to each element $x \in X$ (the Domain) exactly one element $y \in Y$ (the Codomain). The set of all actual mapped values $f(X) \subseteq Y$ is called the Range.

To analyze the behavior and invertibility of functional transformations in finance, we classify mappings into three fundamental categories:

  • Injective (One-to-One): $f$ is injective if distinct domain elements map to distinct codomain elements: $$\forall x_1, x_2 \in X, \quad f(x_1) = f(x_2) \implies x_1 = x_2$$ An injective mapping is strictly required for the existence of a unique inverse function $f^{-1}(y)$.
  • Surjective (Onto): $f$ is surjective if every element in the codomain $Y$ is mapped to by at least one element in the domain: $$\forall y \in Y, \quad \exists x \in X \quad \text{s.t.} \quad f(x) = y$$ This means the Range is exactly equal to the Codomain ($f(X) = Y$).
  • Bijective: A function that is both injective and surjective. A bijective mapping has a perfect, unique one-to-one correspondence, guaranteeing that a unique, stable inverse transform $f^{-1}: Y \to X$ exists across the entire range.

Real-World Quantitative Finance Examples:

• Many-to-One Payoffs (Symmetric Payoffs): An option straddle payoff $V(S) = |S - K|$ maps two different spot prices (e.g. $S_1 = K + \delta S$ and $S_2 = K - \delta S$) to the exact same payoff value $\delta S$. This makes it non-invertible.

• One-to-Many Mappings (Implied Volatility Branches): When solving for implied volatility $\sigma$ from option prices, extreme option boundaries or non-monotonic local volatility grids can create multiple branches (One-to-Many). We must apply strict physical domain boundaries to select the singular active branch.

Even vs. Odd Functions & Integration Shortcuts

Functional symmetry yields powerful simplifications when performing definite integrals:

  • Even Functions: Symmetric over the $Y$-axis: $f(-x) = f(x)$ (e.g. $\cos x$, $x^2$). $$\int_{-a}^{a} f(x) dx = 2 \int_{0}^{a} f(x) dx$$
  • Odd Functions: Symmetric over the origin: $f(-x) = -f(x)$ (e.g. $\sin x$, $x^3$). $$\int_{-a}^{a} f(x) dx = 0$$

Worked Solutions & Proofs

Exercise 1: Rigorous Injectivity Proof for $f(x) = 3x + 1$
  1. State the Injectivity Condition: To prove injectivity, we assume $f(x_1) = f(x_2)$ and show that it must logically imply $x_1 = x_2$.
  2. Substitute the algebraic rule: $$f(x_1) = f(x_2) \implies 3x_1 + 1 = 3x_2 + 1$$
  3. Subtract 1 from both sides: $$3x_1 = 3x_2$$
  4. Divide by 3: $$x_1 = x_2$$ Since $f(x_1) = f(x_2) \implies x_1 = x_2$, the function is rigorously proven to be injective. $\blacksquare$
Exercise 2: Deriving Option Straddle Non-Injectivity
  1. Let strike price $K = 100$.
  2. Choose two distinct domain elements: $S_1 = 105$ and $S_2 = 95$ ($S_1 \neq S_2$).
  3. Evaluate payoffs: $$V(S_1) = |105 - 100| = |5| = 5$$ $$V(S_2) = |95 - 100| = |-5| = 5$$
  4. Since $S_1 \neq S_2$ but $V(S_1) = V(S_2)$, the payoff is many-to-one, confirming it is non-injective and therefore has no unique inverse function without branch partition.
