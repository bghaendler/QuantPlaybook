Differential Equations - Introduction

Core classifications, ordinary vs. partial vs. stochastic models, order and degree, and constants of integration.

1. Session Overview & Distinctions

In quantitative finance, differential equations are unavoidable because asset pricing, option valuations, and portfolio risk channels involve pricing assets that change with respect to underlying variables such as asset prices or time.

There are three main types of differential equations:

  • Ordinary Differential Equations (ODEs): Contain ordinary derivatives such as $dy/dx$. They involve functions of a single independent variable.
  • Partial Differential Equations (PDEs): Contain partial derivatives and involve unknown functions of several variables (e.g. Black-Scholes PDE).
  • Stochastic Differential Equations (SDEs): Contain a random term (Wiener process) modeling random market deviations (e.g. Geometric Brownian Motion).

2. Order and Degree Classification

Every ODE has two primary classification metrics:

  • Order: Determined by the highest derivative present in the equation.
  • Degree: The power to which that highest-order derivative is raised.

Any differential equation of degree greater than one is nonlinear, while degree one equations can be linear.

Classification Examples:

Differential Equation Order Degree Linearity
$\frac{dy}{dx} + 5y = x^2$ 1 1 Linear
$\left(\frac{d^2y}{dx^2}\right)^3 + y\frac{dy}{dx} = e^x$ 2 3 Nonlinear (Degree 3, y coefficient)
$\frac{d^2y}{dx^2} + \sqrt{\frac{dy}{dx}} = y$ 2 1 (after squaring) Nonlinear (Fractional power)

3. Working Backwards: Constants of Integration

If we start with a function $y$ that depends on $x$ and $n$ arbitrary constants, differentiating $n$ times produces $n+1$ equations. Eliminating the constants yields an $n^{\text{th}}$-order ODE. This proves that an $n^{\text{th}}$-order ODE has a general solution containing exactly $n$ constants.

Worked Example 1: One Constant (First Order ODE)

Start with: $y = C x$

Differentiate once: $\frac{dy}{dx} = C$

Eliminate $C$ by substituting $C = y/x$:

$$\frac{dy}{dx} = \frac{y}{x} \implies x \frac{dy}{dx} - y = 0 \quad \text{(First-order ODE)}$$

Worked Example 2: Two Constants (Second Order ODE)

Start with: $y = A \cos x + B \sin x$

Differentiate twice:

$$\frac{dy}{dx} = -A \sin x + B \cos x$$

$$\frac{d^2y}{dx^2} = -A \cos x - B \sin x = - (A \cos x + B \sin x)$$

$$\frac{d^2y}{dx^2} + y = 0 \quad \text{(Second-order ODE)}$$

4. Supplementary Conditions: IVP vs. BVP

A general solution contains undetermined constants. To obtain a particular solution, we must impose additional conditions:

  • Initial Value Problems (IVPs): Consist of an $n^{\text{th}}$-order ODE together with $n$ initial conditions specifying the values of $y$ and its derivatives all at the same point $x_0$ (e.g. $y(0) = 1, y'(0) = 0$).
  • Boundary Value Problems (BVPs): Consist of an $n^{\text{th}}$-order ODE together with $n$ boundary conditions specifying the values of $y$ at $n$ different points (e.g. $y(0) = 0, y(1) = 5$).

Initial conditions and boundary conditions cannot be mixed within a single problem, and a valid solution must satisfy both the differential equation and every one of the conditions.
