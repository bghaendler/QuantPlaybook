Maths Primer 4. Second Order ODEs P2

Second Order Ordinary Differential Equations - Constant Coefficients

Auxiliary equations, discriminants, and the three distinct solution cases (real, repeated, and complex roots).

1. The Auxiliary Equation Derivation

Consider the second-order homogeneous linear ODE with constant coefficients $a$, $b$, and $c$:

$$a y'' + b y' + c y = 0$$

Motivated by the first-order solution form, we try a trial solution of exponential form: $y = e^{\lambda x}$. Differentiating once and twice yields:

$$y' = \lambda e^{\lambda x} \quad \text{and} \quad y'' = \lambda^2 e^{\lambda x}$$

Substituting these back into the differential equation gives:

$$a \lambda^2 e^{\lambda x} + b \lambda e^{\lambda x} + c e^{\lambda x} = 0 \implies e^{\lambda x} \left( a \lambda^2 + b \lambda + c \right) = 0$$

Since the exponential function $e^{\lambda x}$ is never zero, we obtain the Auxiliary (Characteristic) Equation:

$$a \lambda^2 + b \lambda + c = 0$$

2. The Three Root Cases

The solution of the auxiliary equation via the quadratic formula leads to three distinct cases depending on the sign of the discriminant $b^2 - 4ac$:

Case 1: $b^2 > 4ac$ (Two Distinct Real Roots $\lambda_1, \lambda_2$)

The general solution is a linear combination of two distinct exponential curves:

$$y = c_1 e^{\lambda_1 x} + c_2 e^{\lambda_2 x}$$

Example: $y'' - 3y' - 4y = 0 \implies \lambda^2 - 3\lambda - 4 = 0 \implies \lambda_1 = 4, \lambda_2 = -1$. General solution is: $y = c_1 e^{4x} + c_2 e^{-x}$.

Case 2: $b^2 = 4ac$ (One Repeated Real Root $\lambda = -b/2a$)

A second independent solution is found by multiplying by $x$. The general solution is:

$$y = (c_1 + c_2 x) e^{\lambda x}$$

Example: $y'' - 8y' + 16y = 0 \implies \lambda^2 - 8\lambda + 16 = 0 \implies \lambda = 4$ (repeated). General solution: $y = e^{4x}(c_1 + c_2 x)$.

Case 3: $b^2 < 4ac$ (Complex Conjugate Roots $p \pm iq$)

Using Euler's identity ($e^{\pm i\theta} = \cos\theta \pm i\sin\theta$), the general solution is expressed in trigonometric terms:

$$y = e^{px} (A \cos qx + B \sin qx)$$

Example: $y'' - 3y' + 5.25y = 0 \implies \lambda^2 - 3\lambda + 5.25 = 0 \implies \lambda = 1.5 \pm i\sqrt{3}$. Solution: $y = e^{1.5x}(A \cos\sqrt{3}x + B \sin\sqrt{3}x)$.
