Properties of Vectors

Vector definition in $\mathbb{R}^n$, vector arithmetic, dot products, orthogonality, and concept of length (norms).

1. Vector Definitions & Equality

Real $n$-dimensional vectors belong to the set $\mathbb{R}^n$. An $n$-tuple is defined as:

$$\underline{v} = (v_1, v_2, \dots, v_n) \in \mathbb{R}^n$$

where $\underline{v}$ is a vector of dimension $n$, and the elements $v_i$ ($i = 1, \dots, n$) are called the components of $\underline{v}$.

Any pair of vectors $\underline{u}, \underline{v} \in \mathbb{R}^n$ are equal if and only if their corresponding components are equal ($u_i = v_i$ for all $i$).

Plain English Notes

The Big Picture: A vector is an arrow pointing in space, or a list of coordinates telling you how to get from a starting point (the origin) to an endpoint.

  • Dimensionality: A 2D vector has $(x, y)$ coordinates; a 3D vector has $(x, y, z)$. In finance, an $N$-dimensional vector could represent the weights of $N$ assets in a portfolio.
  • Equality: Two vectors are identical only if they point in the exact same direction and have the exact same length (components match one-for-one).

2. Vector Arithmetic & Dot Product

Let $\underline{u}, \underline{v} \in \mathbb{R}^n$ and $k \in \mathbb{R}$ be a scalar:

Vector Addition: $$\underline{u} + \underline{v} = (u_1 + v_1, u_2 + v_2, \dots, u_n + v_n)$$ Scalar Multiplication: $$k\underline{u} = (ku_1, ku_2, \dots, ku_n)$$ Dot (Scalar) Product: $$\underline{u} \cdot \underline{v} = \sum_{i=1}^n u_i v_i \in \mathbb{R}$$ Orthogonality: $$\underline{u} \cdot \underline{v} = 0 \quad \text{(perpendicular)}$$

Plain English Notes

The Big Picture: Adding vectors means placing them head-to-tail to find the combined shortcut path. Multiplying by a scalar changes the length (stretching/shrinking) or reverses direction.

  • Dot Product: Measures how much two vectors align. If they point in the same direction, it's positive. If they point in opposite directions, it's negative.
  • Orthogonality: If they are perpendicular, the dot product is exactly zero, meaning they share no common direction at all. In finance, this indicates uncorrelated assets.

3. Concept of Length (Norm) in $\mathbb{R}^n$

For any vector $\underline{v} = (v_1, \dots, v_n)$, its length is called the norm, denoted as $\|\underline{v}\|$:

Euclidean Norm: $$\|\underline{v}\| = \sqrt{(v_1)^2 + (v_2)^2 + \dots + (v_n)^2}$$ Distance between vectors: $$\|\underline{v} - \underline{u}\| = \sqrt{(v_1 - u_1)^2 + \dots + (v_n - u_n)^2}$$ Angle Definition: $$\cos\theta = \frac{\underline{u} \cdot \underline{v}}{\|\underline{u}\| \|\underline{v}\|}$$

Plain English Notes

The Big Picture: The norm is simply the straight-line length of the vector from the origin, calculated using the Pythagorean theorem.

  • Unit Vectors: Vectors scaled to have a length of exactly $1$. They are useful for representing directions without worrying about magnitude.
  • Portfolio Volatility: The variance of a portfolio is computed using vector norms and matrix multiplications; minimizing this norm is how we find the "minimum variance portfolio."
