Properties of Vectors

Vector definition in $\mathbb{R}^n$, vector arithmetic, dot products, orthogonality, and concept of length (norms).

1. Vector Definitions & Equality

Real $n$-dimensional vectors belong to the set $\mathbb{R}^n$. An $n$-tuple is defined as:

$$\underline{v} = (v_1, v_2, \dots, v_n) \in \mathbb{R}^n$$

where $\underline{v}$ is a vector of dimension $n$, and the elements $v_i$ ($i = 1, \dots, n$) are called the components of $\underline{v}$.

Any pair of vectors $\underline{u}, \underline{v} \in \mathbb{R}^n$ are equal if and only if their corresponding components are equal ($u_i = v_i$ for all $i$).

2. Vector Arithmetic & Dot Product

Let $\underline{u}, \underline{v} \in \mathbb{R}^n$ and $k \in \mathbb{R}$ be a scalar:

Vector Addition:

Only valid if the dimensions of both vectors are identical.

$$\underline{u} + \underline{v} = (u_1 + v_1, u_2 + v_2, \dots, u_n + v_n)$$ Scalar Multiplication: $$k\underline{u} = (ku_1, ku_2, \dots, ku_n)$$ Dot (Scalar) Product:

The dot product is a commutative operation resulting in a scalar quantity:

$$\underline{u} \cdot \underline{v} = u_1v_1 + u_2v_2 + \dots + u_nv_n \in \mathbb{R}$$ $$\underline{u} \cdot \underline{v} = \underline{v} \cdot \underline{u}$$ Orthogonality:

If the dot product of two vectors is zero, they are said to be orthogonal (geometrically perpendicular):

$$\underline{u} \cdot \underline{v} = 0$$

3. Concept of Length (Norm) in $\mathbb{R}^n$

In 2-Dimensions (2-D):

For a vector $\underline{u} = (x_1, y_1)$:

$$|\underline{u}| = \sqrt{(x_1)^2 + (y_1)^2} \quad \text{and} \quad \theta = \arctan\left(\frac{y_1}{x_1}\right)$$ $$\text{Unit vector } \underline{\hat{u}} = \frac{\underline{u}}{|\underline{u}|} \implies \underline{u} = |\underline{u}| \underline{\hat{u}} \quad (\text{where } |\underline{\hat{u}}| = 1)$$ $$\text{Distance between } \underline{u}, \underline{v} \in \mathbb{R}^2: \quad |\underline{v} - \underline{u}| = \sqrt{(v_1 - u_1)^2 + (v_2 - u_2)^2}$$ In 3-Dimensions (3-D): $$|\underline{v}| = \sqrt{(x_1)^2 + (y_1)^2 + (z_1)^2}$$ Generalization to $\mathbb{R}^n$:

For any vector $\underline{v} = (v_1, \dots, v_n)$, its length is called the norm, denoted as $\|\underline{v}\|$:

$$\|\underline{v}\| = \sqrt{(v_1)^2 + (v_2)^2 + \dots + (v_n)^2}$$ $$\text{Distance: } \quad \|\underline{v} - \underline{u}\| = \sqrt{(v_1 - u_1)^2 + \dots + (v_n - u_n)^2}$$ $$\text{Angle Definition: } \quad \underline{u} \cdot \underline{v} = \|\underline{u}\| \|\underline{v}\| \cos\theta$$
