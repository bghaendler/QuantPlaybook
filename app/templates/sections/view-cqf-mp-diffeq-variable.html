Maths Primer 7. Variable Coeffs & Euler

Linear ODEs with Variable Coefficients - Cauchy-Euler Equations

Variable coefficient structures, trial solutions $x^\lambda$, three roots cases, and logarithmic transformations to constant coefficients.

1. Cauchy-Euler Equations

A Cauchy-Euler equation is a specific type of variable-coefficient ordinary differential equation where the coefficient of each derivative term is a polynomial whose degree matches the order of that derivative. The second-order form is:

$$A x^2 \frac{d^2 y}{dx^2} + B x \frac{dy}{dx} + C y = 0$$

To solve this, we assume a trial solution of the form: $y = x^{\lambda}$. Differentiating once and twice yields:

$$\frac{dy}{dx} = \lambda x^{\lambda-1} \quad \text{and} \quad \frac{d^2y}{dx^2} = \lambda(\lambda-1)x^{\lambda-2}$$

Substituting these back into the differential equation gives:

$$A x^2 \left( \lambda(\lambda-1)x^{\lambda-2} \right) + B x \left( \lambda x^{\lambda-1} \right) + C x^\lambda = 0 \implies x^\lambda \left[ A \lambda(\lambda-1) + B \lambda + C \right] = 0$$

Assuming $x \ne 0$, we obtain the **Auxiliary Equation**:

$$A \lambda^2 + (B - A)\lambda + C = 0$$

Note: This is a key distinction from the constant coefficient case; the middle coefficient becomes $(B - A)$ due to the second differentiation product $x^2 \cdot x^{\lambda-2}$.

2. The Three Root Cases

Case 1: Two Distinct Real Roots $\lambda_1, \lambda_2$

The general solution is a linear combination of power functions:

$$y = c_1 x^{\lambda_1} + c_2 x^{\lambda_2}$$

Example: For $A=1, B=-2, C=-4 \implies \lambda^2 - 3\lambda - 4 = 0 \implies \lambda = 4, -1$. Solution: $y = c_1 x^4 + c_2 x^{-1}$.

Case 2: One Repeated Real Root $\lambda$

A second independent solution is found by multiplying by $\log x$. The general solution is:

$$y = x^\lambda \left( c_1 + c_2 \log x \right)$$

Example: For $A=1, B=-7, C=16 \implies \lambda^2 - 8\lambda + 16 = 0 \implies \lambda = 4$ (repeated). Solution: $y = x^4 (c_1 + c_2 \log x)$.

Case 3: Complex Conjugate Roots $\alpha \pm i\beta$

Rewriting $x^{\pm i\beta} = e^{\pm i\beta \log x}$ and applying Euler's formula produces trigonometric terms with logarithmic arguments:

$$y = x^\alpha \left[ A \cos(\beta \log x) + B \sin(\beta \log x) \right]$/p>

Example: For $A=1, B=-3, C=13 \implies \lambda^2 - 4\lambda + 13 = 0 \implies \lambda = 2 \pm 3i$. Solution: $y = x^2 \left[ A \cos(3 \log x) + B \sin(3 \log x) \right]$.

3. Logarithmic Transformation to Constant Coefficients

Non-homogeneous Cauchy-Euler equations can be converted into constant-coefficient problems using the change of variable: $t = \log x \implies x = e^t$. Applying the chain rule to represent derivatives gives:

First Derivative Transformation: $$\frac{dy}{dx} = \frac{dy}{dt}\frac{dt}{dx} = \frac{1}{x}\frac{dy}{dt} \implies x \frac{dy}{dx} = \frac{dy}{dt}$$ Second Derivative Transformation: $$\frac{d^2y}{dx^2} = \frac{d}{dx}\left[ \frac{1}{x}\frac{dy}{dt} \right] = -\frac{1}{x^2}\frac{dy}{dt} + \frac{1}{x}\frac{d}{dx}\left[ \frac{dy}{dt} \right]$$ $$= -\frac{1}{x^2}\frac{dy}{dt} + \frac{1}{x^2}\frac{d^2y}{dt^2} = \frac{1}{x^2} \left( \frac{d^2y}{dt^2} - \frac{dy}{dt} \right)$$

$$\implies x^2 \frac{d^2y}{dx^2} = \frac{d^2y}{dt^2} - \frac{dy}{dt}$$

Substituting these expressions into the Euler equation completely cancels the variable $x$ terms, leaving a constant-coefficient ordinary differential equation in $y$ as a function of $t$.
