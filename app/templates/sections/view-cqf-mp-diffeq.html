Differential Equations model how quantities evolve continuously. This cheat sheet consolidates key concepts and analytical solution methods: first-order separable and linear ODEs, second-order constant-coefficient systems, non-homogeneous trials, Cauchy-Euler variable coefficient equations, and introduces the PDEs and SDEs governing financial derivatives.

1. First-Order Ordinary Differential Equations

First-order ODEs involve only the first derivative $y'$. Analytical solution methods depend on the equation structure:

  • Separable Equations: If $\frac{dy}{dx} = f(x)g(y)$, separate variables and integrate: $$\int \frac{1}{g(y)} dy = \int f(x) dx + C$$
  • Linear First-Order ODEs: In the standard form $y' + P(x)y = Q(x)$, we solve using an Integrating Factor $I(x) = e^{\int P(x) dx}$: $$\frac{d}{dx}[I(x)y] = I(x)Q(x) \implies y(x) = \frac{1}{I(x)} \left(\int I(x)Q(x) dx + C\right)$$
  • Exact Equations: $M(x,y)dx + N(x,y)dy = 0$ is exact if $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$, implying a potential function $\Psi(x,y) = C$ exists.

2. Second-Order Constant Coefficient Homogeneous ODEs

Linear second-order homogeneous ODEs with constant coefficients take the form:

$$ay'' + by' + cy = 0$$

We assume a solution of the form $y = e^{rx}$, leading to the quadratic auxiliary (characteristic) equation $ar^2 + br + c = 0$. The roots $r_1, r_2$ yield three distinct general solution structures:

  • Distinct Real Roots ($\Delta > 0$): Two real roots $r_1, r_2$: $$y = A e^{r_1 x} + B e^{r_2 x}$$
  • Repeated Root ($\Delta = 0$): One repeated root $r = -b/2a$: $$y = (A + Bx)e^{rx}$$
  • Complex Conjugate Roots ($\Delta < 0$): Roots $r = \alpha \pm i\beta$ (where $\alpha = -b/2a$, $\beta = \sqrt{-\Delta}/2a$): $$y = e^{\alpha x}(A \cos\beta x + B \sin\beta x)$$

3. Non-Homogeneous ODEs & Undetermined Coefficients

For non-homogeneous equations $ay'' + by' + cy = f(x)$, the general solution is the sum of the complementary function $y_c$ (solving the homogeneous LHS) and a particular integral $y_p$ (solving the non-homogeneous RHS): $y = y_c + y_p$.

Method of Undetermined Coefficients: We guess the form of $y_p$ based on the shape of $f(x)$:

RHS function $f(x)$ Particular Trial $y_p(x)$
Polynomial: $p_n(x)$ $A_n x^n + \dots + A_1 x + A_0$
Exponential: $e^{kx}$ $C e^{kx}$
Trig: $\cos(\omega x)$ or $\sin(\omega x)$ $D \cos(\omega x) + E \sin(\omega x)$

λ CRITICAL FAILURE CASE: If any term in the guessed trial solution $y_p(x)$ is already present in the complementary function $y_c(x)$ (i.e. is a homogeneous solution), the trial fails! You must multiply the trial by $x$ (or $x^2$ for double repeated roots) to restore linear independence.

4. Variable Coefficients & Cauchy-Euler Equations

When coefficients depend on $x$, analytical solutions are harder to find. The notable exception is the Cauchy-Euler Equation:

$$a x^2 y'' + b x y' + c y = 0$$

We assume a solution of the form $y = x^m \implies y' = m x^{m-1} \implies y'' = m(m-1) x^{m-2}$. Substituting these in yields the auxiliary equation:

$$a m(m-1) + b m + c = 0 \implies a m^2 + (b - a)m + c = 0$$

The roots $m_1, m_2$ determine the solution structure:

  • Distinct Real Roots: $y = A x^{m_1} + B x^{m_2}$
  • Repeated Real Root ($m$): $y = (A + B \ln x)x^m$
  • Complex Roots ($\alpha \pm i\beta$): $y = x^\alpha [A \cos(\beta \ln x) + B \sin(\beta \ln x)]$

Alternatively, this can be transformed into a constant-coefficient equation by substituting $x = e^t$ ($t = \ln x$) and applying the chain rule.

5. From ODEs to Partial (PDE) & Stochastic (SDE) Differential Equations

In quantitative finance, assets fluctuate randomly rather than smoothly. To model this, we transition from ODEs to **Stochastic Differential Equations (SDEs)**. A standard SDE (such as Geometric Brownian Motion) is formulated as:

$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

where $dW_t$ is a Brownian increment. Integrating SDEs requires **Itô's Lemma** (the stochastic equivalent of Taylor series expansion).

The Black-Scholes PDE: By constructing a risk-free portfolio matching derivative and underlying movements, we eliminate $dW_t$, yielding the famous deterministic Black-Scholes partial differential equation (PDE):

$$\frac{\partial V}{\partial t} + r S \frac{\partial V}{\partial S} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} - rV = 0$$

The Feynman-Kac Connection The Feynman-Kac formula bridges the gap between SDEs (probability/expectations) and PDEs (deterministic boundary value problems). It states that the solution to a parabolic PDE of the form: $$\frac{\partial u}{\partial t} + \mu(x,t)\frac{\partial u}{\partial x} + \frac{1}{2}\sigma^2(x,t)\frac{\partial^2 u}{\partial x^2} - r u = 0$$ with terminal condition $u(x,T) = \psi(x)$ can be written as a conditional expectation: $$u(x,t) = \mathbb{E}_t \left[ e^{-r(T-t)} \psi(X_T) \,\Big|\, X_t = x \right]$$ where $X_t$ follows the SDE $dX_t = \mu(X_t, t)dt + \sigma(X_t, t)dW_t$. This proves that option pricing can be solved **either** by finding analytical solutions to the BS-PDE or by simulating asset path averages (Monte Carlo).
