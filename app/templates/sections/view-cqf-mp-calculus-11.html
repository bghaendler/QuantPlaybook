1.1 Mathematical Shorthand & Number Systems

Rigorous mathematical notation, logic symbols, and set descriptions used in quantitative finance.

1.1.1 Logic Shorthand Notation

Mathematical proofs and derivations utilize concise symbolic logic shorthand to represent statements efficiently. In continuous asset path modeling and probability fields, these symbols form the building blocks of rigorous theorems:

∃ : there exists (existential quantifiers)
∀ : for all (universal quantifiers)
∴ : therefore (logical consequence)
∵ : because (logical premise)
⇒ : which gives / implies
s.t. : such that (conditional clause)
⇔ : if and only if (equivalence)
∈ : element of (set membership)

E.g. We write the statement "For all real volatility parameters $\sigma$, there exists a unique asset price $S$ such that the Black-Scholes PDE is satisfied" as:

$\forall \sigma \in \mathbb{R}^+, \exists ! S \text{ s.t. } \frac{\partial V}{\partial t} + r S \frac{\partial V}{\partial S} + \frac{1}{2} \sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} - rV = 0$.

1.1.2 Number Systems & Fields

Real-world quant modeling relies on specific number fields designed to represent different financial concepts:

  • Natural Numbers ($\mathbb{N}$): $\{0, 1, 2, 3, \dots\}$ — Used to index discrete monitoring dates or steps $t_k$ in Monte Carlo simulations.
  • Integers ($\mathbb{Z}$): $\{0, \pm 1, \pm 2, \dots\}$ — Represents countable asset state units, contract sizes, or portfolio position holdings.
  • Rationals ($\mathbb{Q}$): $\{p/q : p,q \in \mathbb{Z}\}$ — Represents discrete price grid spaces in Finite Difference Schemes.
  • Irrationals ($\mathbb{Q}$'): $\{\sqrt{2}, \pi, e\}$ — Drive continuous distributions. Euler's number $e$ forms the foundation of continuous discounting and Brownian drift.
  • Reals ($\mathbb{R}$): Union of all rational and irrational fields. Volatilities, spot prices, interest rates, and stochastic paths reside here.
  • Complex ($\mathbb{C}$): $\{x + iy : i = \sqrt{-1}\}$ — Forms the mathematical backbone of Characteristic Functions, Fourier Inversion, and FFT pricing methods (e.g. Heston Model).
