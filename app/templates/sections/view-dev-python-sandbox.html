How It Works

        
    
            
  • Pyodide is the CPython interpreter compiled to WebAssembly (Emscripten), running in the browser's sandbox with no installation and no server round-trips.
    • 
            
  • The scientific stack (NumPy, SciPy, pandas, matplotlib) is ported; packages load on demand from the CDN via pyodide.loadPackage().
    • 
            
  • State persists between runs in this session — define a function in one run, call it in the next.
    • 
            
  • Limits: no native threads or raw sockets, ~2 GB memory, and heavy first-load — for production research, this complements (not replaces) a local environment.
