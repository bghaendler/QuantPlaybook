Definitions of experiments, event subsets, sample spaces, and likelihood boundaries.

1.1.1 Probability Definitions & Boundaries

Before computing complex pricing odds, we define the structural foundations of random systems:

  • Experiment: A repeatable process (e.g. coin toss, portfolio shock) that gives rise to a set of outcomes.
  • Event ($E$): A collection (or subset) of one or more outcomes.
  • Sample Space ($\Omega$): The exhaustive set of all possible outcomes of an experiment.

The probability scale is strictly bounded by absolute certainty (1) and absolute impossibility (0):

$$0 \le P(E) \le 1$$

The Complementary Event ($E'$): An event either occurs or it does not. The probability of the complement (not $E$) is defined as:

$$P(E') = 1 - P(E)$$
