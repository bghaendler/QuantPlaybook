Asset Returns Stylised Facts

Log Returns, Leptokurtosis, Autocorrelations, Diurnal U-Shapes, and Realized Volatility

1. Linear vs. Continuously Compounded Returns

To model financial assets over time, we evaluate two primary return definitions:

  • Simple Return ($r'_t$): Measures the percentage price change: $$r'_t = \frac{p_t + d_t - p_{t-1}}{p_{t-1}}$$ Simple returns aggregate linearly across a portfolio: $r_{p} = \sum w_i r'_i$.
  • Continuously Compounded Return ($r_t$): Defined as the natural log of price relatives: $$r_t = \ln(p_t + d_t) - \ln(p_{t-1})$$ Log-returns are additive over time: $r_{0 \to T} = \sum_{t=1}^T r_t$.
💡 Taylor Series Equivalence: When returns are small ($r_t \approx 0$), simple and log-returns are numerically similar: $r'_t = e^{r_t}-1 \approx r_t + \frac{1}{2}r_t^2 + \dots$

2. Volatility Clustering

Financial return series universally exhibit volatility clustering (Mandelbrot, 1963):

"Large price changes tend to be followed by large changes – of either sign – and small price changes tend to be followed by small changes."

Implication: Daily asset returns cannot be modeled as independent normal distributions. Instead, they behave like a mixture of normals with time-varying variance, which leads directly to leptokurtosis (fat tails).

Three Stylised Facts of Daily Asset Returns

Daily returns across virtually all stock markets, commodities, and currency pairs exhibit three invariant empirical properties:

1. Heavy Tails (Non-Normality)

Daily returns are non-normal. They display symmetry but are highly leptokurtic (fat tails and high narrow peaks). Extreme moves ($>4\sigma$) occur $1$-$2$ times a year (under standard normal, only once every 60 years).

2. Zero Autocorrelation

Raw returns show almost zero linear autocorrelation over daily intervals. You cannot easily predict direction or returns using linear historical relationships. Autocorrelations behave like i.i.d. noise: $\hat{\rho}_\tau \sim N(0, 1/n)$.

3. Volatility Persistence

While raw returns show no autocorrelation, absolute returns $|r_t|$ and squared returns $r_t^2$ show highly persistent, significant positive autocorrelation that decays extremely slowly over months.

1. Intraday High-Frequency Properties

When price data is sampled at intermediate intraday intervals (e.g. 5 minutes) rather than daily, several distinct econometric properties manifest:

1. Extremes and Reverse CLT

As the sampling interval shortens, observed excess kurtosis increases dramatically. Returns display increasingly non-normal, leptokurtic distributions—representing the Central Limit Theorem operating in reverse.

2. Negative Autocorrelation (Bid-Ask Bounce)

Over short intervals (seconds/ticks), consecutive returns show artificial negative serial correlation at lag one. This is driven by transactions bouncing between bid and ask quotes.

3. 24-Hour Volatility Peaks

autocorrelation of absolute returns peaks sharp at exact integer multiples of 24 hours (e.g. lag 48 for 30-min data), demonstrating that intraday volatility levels are highly predictable from previous days at the same time.

Realized Volatility ($\sigma_t$) and Properties

Because high-frequency data provides a significantly larger number of observations, volatility can be estimated far more precisely than with daily closing prices. Let $r_{t,j}$ ($j=1, \dots, N$) denote the intraday high-frequency returns on day $t$. The Realized Volatility ($\sigma_t$) is defined as:

$$\sigma_t = \sqrt{\sum_{j=1}^{N} r_{t,j}^2}$$

Three Fundamental Empirical Properties:

1. Standardization Normality
$$\frac{r_t}{\sigma_t} \sim N(0, 1)$$ Standardizing daily returns by their realized volatility results in an approximately standard normal distribution.
2. Log-Normal Distribution
$$\ln(\sigma_t) \sim \text{Gaussian}$$ The distribution of realized volatility ($\sigma_t$) over time is approximately lognormal.
3. Long Memory Persistence
$$\text{Corr}(\sigma_t, \sigma_{t+\tau}) \gg 0$$ The autocorrelation of realized volatility decays very slowly over time, behaving like a long memory process.
