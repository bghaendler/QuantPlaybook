Properties of Matrices

Matrix layouts, arithmetic, matrix multiplication rules, symmetric/skew-symmetric transposes, and determinants.

1. Matrix Definitions & Layout

A matrix is a rectangular array $A = (a_{ij})$ for $i = 1, \dots, m$ and $j = 1, \dots, n$, written as:

$$A = \begin{pmatrix} a_{11} & a_{12} & \dots & a_{1n} \\ a_{21} & a_{22} & \dots & a_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ a_{m1} & a_{m2} & \dots & a_{mn} \end{pmatrix}$$

This is an $(m \times n)$ matrix (having $m$ rows and $n$ columns). If $m = n$, the matrix is called a square matrix. The product $mn$ yields the total number of elements in the matrix.

2. Matrix Arithmetic & Multiplication

Let $A, B \in {^m\mathbb{R}^n}$:

Matrix Addition:

Matrices can only be added if they are of the same dimensions. Corresponding elements are added:

$$A + B = (a_{ij} + b_{ij}) = B + A$$ Matrix Multiplication:

To multiply two square matrices $\mathbf{A}$ and $\mathbf{B}$ such that $\mathbf{C} = \mathbf{AB}$, the elements of $\mathbf{C}$ are found using the relation:

$$C_{ij} = \sum_{k=1}^N A_{ik}B_{kj}$$

This means that the $i^{\text{th}}$ row of $\mathbf{A}$ is dotted with the $j^{\text{th}}$ column of $\mathbf{B}$:

$$\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} e & f \\ g & h \end{pmatrix} = \begin{pmatrix} ae+bg & af+bh \\ ce+dg & cf+dh \end{pmatrix}$$

CRITICAL NOTE: Matrix multiplication is non-commutative: $\mathbf{AB} \neq \mathbf{BA}$.

Dimension compatibility rule: $A_{p \times n} B_{n \times m} \to C_{p \times m}$.

3. Matrix Transpose

The transpose of a matrix with entries $A_{ij}$ is the matrix with entries $A_{ji}$, denoted as $\mathbf{A}^{\mathrm{T}}$. The rows of the original matrix become the columns of the transposed matrix:

  • If $\mathbf{A} = \mathbf{A}^{\mathrm{T}}$, the matrix is called symmetric.
  • If $a_{ij} = -a_{ji}$ with $a_{ii} = 0$, the matrix is called skew-symmetric.
Properties of Transposition:
  1. $(\mathbf{A} + \mathbf{B})^{\mathrm{T}} = \mathbf{A}^{\mathrm{T}} + \mathbf{B}^{\mathrm{T}}$
  2. $(\mathbf{A}^{\mathrm{T}})^{\mathrm{T}} = \mathbf{A}$
  3. $(k\mathbf{A})^{\mathrm{T}} = k\mathbf{A}^{\mathrm{T}}$ (where $k$ is a scalar)
  4. $(\mathbf{AB})^{\mathrm{T}} = \mathbf{B}^{\mathrm{T}}\mathbf{A}^{\mathrm{T}}$   (Order reverses!)

4. Matrix Representation of Linear Equations

Consider a $2 \times 2$ set of linear equations for the unknowns $x$ and $y$:

$$ax + by = p \quad \text{and} \quad cx + dy = q$$

We can write this system in matrix-vector form as:

$$\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} p \\ q \end{pmatrix} \quad \implies \quad \mathbf{A}\underline{x} = \underline{p}$$

This system can be solved uniquely if the coefficient matrix $\mathbf{A}$ is invertible. This is true if and only if its determinant is non-zero:

$$\det(A) = \begin{vmatrix} a & b \\ c & d \end{vmatrix} = ad - bc \neq 0$$
