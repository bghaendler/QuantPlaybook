Properties of Matrices

Matrix layouts, arithmetic, matrix multiplication rules, symmetric/skew-symmetric transposes, and determinants.

1. Matrix Definitions & Layout

A matrix is a rectangular array $A = (a_{ij})$ for $i = 1, \dots, m$ and $j = 1, \dots, n$, written as:

$$A = \begin{pmatrix} a_{11} & a_{12} & \dots & a_{1n} \\ a_{21} & a_{22} & \dots & a_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ a_{m1} & a_{m2} & \dots & a_{mn} \end{pmatrix}$$

This is an $(m \times n)$ matrix (having $m$ rows and $n$ columns). If $m = n$, it is a square matrix.

Plain English Notes

The Big Picture: A matrix is just a spreadsheet of numbers. If a single number is a point and a vector is a line, a matrix is a grid of coordinates.

  • Indices: The first index ($i$) is the row (vertical position) and the second index ($j$) is the column (horizontal position).
  • State Vectors: In finance, we group multiple stock returns, prices, or yields into matrices to compute portfolio states in a single operation.

2. Matrix Arithmetic & Multiplication

Let $A, B \in {^m\mathbb{R}^n}$:

Matrix Addition: Same dimensions only. $$A + B = (a_{ij} + b_{ij})$$ Matrix Multiplication (AB): $$C_{ij} = \sum_{k=1}^N A_{ik}B_{kj}$$

Non-Commutative: $\mathbf{AB} \neq \mathbf{BA}$ in general.

Plain English Notes

The Big Picture: Adding matrices is simple: just add matching boxes. Multiplying them is completely different: it dots the rows of the first matrix with the columns of the second matrix.

  • Non-Commutativity: Order matters! Transforming space by matrix $\mathbf{B}$ and then matrix $\mathbf{A}$ (written $\mathbf{AB}$) usually yields a different result than doing them in reverse ($\mathbf{BA}$).
  • Analogy: Putting on socks then shoes is NOT the same as putting on shoes then socks.

3. Matrix Transpose

The transpose of a matrix with entries $A_{ij}$ is the matrix with entries $A_{ji}$, denoted as $\mathbf{A}^{\mathrm{T}}$. The rows of the original matrix become the columns of the transposed matrix:

  • If $\mathbf{A} = \mathbf{A}^{\mathrm{T}}$, the matrix is symmetric.
  • If $a_{ij} = -a_{ji}$ with $a_{ii} = 0$, the matrix is skew-symmetric.
  • Transposing product: $(\mathbf{AB})^{\mathrm{T}} = \mathbf{B}^{\mathrm{T}}\mathbf{A}^{\mathrm{T}}$.

Plain English Notes

The Big Picture: Transposing a matrix is like flipping the grid diagonally. Rows become columns, and columns become rows.

  • Symmetric Matrices: Perfect mirrors across the main diagonal. In quant finance, covariance matrices of asset returns are always symmetric.
  • Reversing Order: In the transpose of a product, $(\mathbf{AB})^{\mathrm{T}} = \mathbf{B}^{\mathrm{T}}\mathbf{A}^{\mathrm{T}}$, the order of multiplication must flip for the dimensions to match up properly.

4. Matrix Representation of Linear Equations

Consider a $2 \times 2$ set of linear equations for the unknowns $x$ and $y$:

$$ax + by = p \quad \text{and} \quad cx + dy = q$$

Written in matrix-vector form:

$$\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} p \\ q \end{pmatrix} \quad \implies \quad \mathbf{A}\underline{x} = \underline{p}$$

This system can be solved uniquely if $\det(A) = ad - bc \neq 0$.

Plain English Notes

The Big Picture: Matrices compress large systems of equations into single equations. Instead of dealing with thousands of variables separately, we can operate on them as unified vectors.

  • Determinant: A scalar indicating whether the matrix operation collapses the spatial dimensions.
  • Invertibility: If the determinant is non-zero, the system has a unique, reversible mapping, meaning we can solve it.
