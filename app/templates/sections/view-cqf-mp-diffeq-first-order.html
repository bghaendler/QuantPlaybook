First Order Ordinary Differential Equations

Simplest forms (missing variables), variable separable equations, and linear ODEs solved using integrating factors.

1. Notational Conventions

A first-order ordinary differential equation relates an independent variable $x$, a dependent variable $y$, and the first derivative $dy/dx$. There are two equivalent ways to represent this relationship:

  • Implicit Form ($F$): All terms gathered on one side: $$F\left(x, y, \frac{dy}{dx}\right) = 0$$
  • Explicit Form ($f$): The derivative $dy/dx$ is isolated on the left: $$\frac{dy}{dx} = f(x, y)$$

These are simply different notational conventions representing the exact same underlying equation.

2. Simple Equations (Absent Variables)

When one of the variables is absent from the right-hand side, the equation reduces to a direct integration form:

  • Variable $y$ is absent: The equation becomes $dy/dx = f(x)$. This is solved by separating $dx$ across and integrating both sides directly: $$y = \int f(x)\,dx + C$$
  • Variable $x$ is absent: The equation becomes $dy/dx = f(y)$. Rearranging to place $dy$ and $y$ on the left and $dx$ on the right: $$\int \frac{1}{f(y)}\,dy = \int 1\,dx = x + C$$

Worked Initial Value Problem (IVP) Example:

Solve: $\frac{dy}{dx} = \cos^2 y$, subject to initial condition $y = \frac{\pi}{4}$ when $x = 2$.

Separate variables:

$$\frac{1}{\cos^2 y}\,dy = dx \implies \sec^2 y\,dy = dx$$

Integrate both sides:

$$\int \sec^2 y\,dy = \int 1\,dx \implies \tan y = x + C$$

Apply the initial condition ($y(\pi/4) \implies x = 2$):

$$\tan\left(\frac{\pi}{4}\right) = 2 + C \implies 1 = 2 + C \implies C = -1$$

$$y = \arctan(x - 1) \quad \text{(Particular solution)}$$

3. Variable Separable Equations

A first-order equation is separable if $f(x,y)$ can be factored into a product $g(x) h(y)$. If so, we can separate all $y$ terms to one side and $x$ terms to the other:

$$\frac{dy}{dx} = g(x)h(y) \implies \int \frac{1}{h(y)}\,dy = \int g(x)\,dx + C$$

For example, $x^2 \sin y$ and $e^x \ln y$ are separable, while $x + y$ is not separable.

Worked Separable Example:

Solve: $\frac{dy}{dx} = \frac{x^2 + 2}{y}$

Separate variables: $y\,dy = (x^2 + 2)\,dx$

Integrate both sides:

$$\int y\,dy = \int (x^2 + 2)\,dx \implies \frac{y^2}{2} = \frac{x^3}{3} + 2x + C_0$$

$$y^2 = \frac{2}{3}x^3 + 4x + C \quad \text{(Implicit general solution)}$$

Note: Leaving a solution in implicit form such as $y^2 = \text{expression}$ is entirely acceptable and preferable to forcing an explicit root form.

4. Linear Equations & Integrating Factor Method

A linear first-order ODE has the standard form:

$$\frac{dy}{dx} + p(x)y = q(x)$$

To solve this, we define an Integrating Factor $R(x)$ as:

$$R(x) = e^{\int p(x)\,dx}$$

Method Proof:

Multiplying both sides of the standard equation by $R(x)$ converts the left-hand side into an exact derivative by product rule:

$$e^{\int p(x)\,dx}\frac{dy}{dx} + p(x)e^{\int p(x)\,dx}y = q(x)e^{\int p(x)\,dx}$$ $$\frac{d}{dx}\left[ y \cdot e^{\int p(x)\,dx} \right] = q(x)e^{\int p(x)\,dx}$$

Now we integrate both sides directly with respect to $x$ and divide through by $R(x)$ to obtain $y$.

CRITICAL REMINDER: The equation must be in standard form (with $1$ as coefficient of $y'$) before identifying $p(x)$.
