1. Mathematical Foundations

Mathematical shorthand, number fields, function mapping types, quadratics, exponentials, and limits.

1.1 Mathematical Shorthand & Number Systems

We establish standard shorthand logic notation frequently used in derivations:

∃ : there exists              ∀ : for all
∴ : therefore              ∵ : because
⇒ : which gives            s.t. : such that
⇔ : if and only if          ≡ : equivalent
∈ : element of             !x : unique x

Standard numbers fields and sets are classified as:

  • Natural Numbers ($\mathbb{N}$): $\{0, 1, 2, 3, \dots\}$ — Used for indexing discrete time-steps $t_k$.
  • Integers ($\mathbb{Z}$): $\{0, \pm 1, \pm 2, \dots\}$ — Represents countable state units or portfolio positions.
  • Rationals ($\mathbb{Q}$): $\{p/q : p,q \in \mathbb{Z}\}$ — E.g. grid spacing steps in Finite Difference Methods.
  • Irrationals ($\mathbb{Q}$'): $\{\sqrt{2}, \pi, e\}$ — Transcendental numbers like Euler's $e$ drive all continuous growth.
  • Reals ($\mathbb{R}$): Union of all rational and irrational fields. Option prices, volatilities, and asset paths belong here.
  • Complex ($\mathbb{C}$): $\{x + iy : i = \sqrt{-1}\}$ — Vital for Fourier Transforms and Characteristic Functions in Heston stochastic volatility models.

Intervals notation: Open $(a,b) \equiv a < x < b$; Closed $[a,b] \equiv a \le x \le b$; Semi-open/closed $[a,b) \equiv a \le x < b$.

1.2 Mapping Theory, Domains & Inverses

A function $f: X \to Y$ maps elements of a domain $X$ uniquely to an image/range set $Y$:

  • One-to-One (1-1): Each $x \in X$ maps to exactly one unique $y \in Y$ (e.g. $y = 3x+1$). Invertible!
  • Many-to-One: Multiple distinct $x_i$ map to the same $y$ (e.g. $y = 2x^2-1$, where $x = \pm 2 \implies y = 7$). Payoff of a straddle or symmetric option.
  • One-to-Many: Maps a single $x$ to multiple $y$ values (e.g., $y = \pm\sqrt{x}$). Note: A one-to-many mapping is multivalued and is NOT a strict mathematical function unless a single branch (e.g. positive) is selected.

Inverse Functions ($f^{-1}(x)$): Maps the image back to the domain. Must satisfy $f(f^{-1}(x)) = x$ and $f^{-1}(f(x)) = x$.

f(x) = 2x² - 1 ⇒ f¯¹(x) = √((x+1)/2)
Domain restrictions exist, e.g. for √((x+1)/2) ⇒ x ≥ -1

Even vs. Odd Functions: Even: $f(-x) = f(x)$ (e.g., $x^2$, $\cos x$). Odd: $f(-x) = -f(x)$ (e.g., $x^3$, $\sin x$). All functions can be written as a sum of even and odd functions.

1.3 Function Types, Implicit Form & Quadratics

Explicit vs. Implicit: Explicit has the dependent variable isolated ($y = f(x)$). Implicit has variables bundled ($f(x,y) = 0$).

Example: $y^2 + x^2 - 16 = 0$ is implicit. Solving gives $y = \pm\sqrt{16-x^2}$ (which is one-to-many, representing two functional branches).

Polynomials: $f(x) = \sum_{k=0}^{n} a_k x^k$ is a polynomial of degree $n$.

Solving the Quadratic Equation $ax^2 + bx + c = 0$: We complete the square to derive the quadratic formula:

$$\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2} \implies x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

  • $b^2 - 4ac > 0$: 2 distinct real roots
  • $b^2 - 4ac = 0$: 1 repeated real root ($x = -b/2a$)
  • $b^2 - 4ac < 0$: Complex conjugate pair roots ($x \in \mathbb{C}$)

1.4 Exponentials, Logarithms & Limit Rules

The number $e$ is defined as the continuous limit: $e = \lim_{n \to \infty} (1 + 1/n)^n \approx 2.718281828$.

The natural logarithm $\ln(x)$ and exponential $e^x$ are mutual inverses ($\ln(e^x) = e^{\ln(x)} = x$).

exey = ex+y, e⁰ = 1
log(xy) = log x + log y
log(x/y) = log x - log y
log(1/x) = -log x

The Modulus Function (Piecewise): $|x| = x$ if $x \ge 0$, and $-x$ if $x < 0$.

Limits and Continuity: A function $f(x)$ is continuous at $x_0$ if $\lim_{x \to x_0} f(x) = f(x_0)$. The limit exists if and only if left-hand and right-hand limits are equal:

$$\lim_{x \to x_0^-} f(x) = \lim_{x \to x_0^+} f(x) = l$$

Exercise 1: Limits & L'Hôpital's Rule

Evaluate the limit $\lim_{x \to 0} \frac{\cos x + 2x - 1}{3x}$ using L'Hôpital's Rule and explain the step-by-step calculus methodology.

Step-by-Step Mathematical Derivation:
  1. Identify the Indeterminate Form: Evaluate at $x = 0$: $$\text{Numerator: } \cos(0) + 2(0) - 1 = 0 \quad , \quad \text{Denominator: } 3(0) = 0$$ This results in the indeterminate form $\frac{0}{0}$. Therefore, L'Hôpital's Rule is applicable.
  2. Differentiate Numerator and Denominator: Let $f(x) = \cos x + 2x - 1 \implies f'(x) = -\sin x + 2$. Let $g(x) = 3x \implies g'(x) = 3$.
  3. Evaluate the Limit of the Derivatives: $$\lim_{x \to 0} \frac{-\sin x + 2}{3} = \frac{0 + 2}{3} = \frac{2}{3}$$
Why do we do this? (Plain English Layman's Walkthrough)

1. What is an indeterminate form? Plugging in zero directly gives $\frac{0}{0}$, which is undefined because division by zero is not mathematically allowed.

2. What is L'Hôpital's Rule? Instead of static values at $x = 0$, we look at the speed (derivative) at which they shrink to zero. The ratio of their speeds gives us their fair limits.

3. Differentiating denominator Differentiating $3x$ yields a constant $3$, removing the division-by-zero crisis.

Exercise 2: Derivation of Continuous Compounding Limit

Prove that the continuous compounding limit holds: $\lim_{n \to \infty} P \left(1 + \frac{r}{n}\right)^{nt} = P e^{rt}$ using logarithmic limits and L'Hôpital's Rule.

Step-by-Step Mathematical Proof:
  1. Isolate the variable part: We want to evaluate the limit $L = \lim_{n \to \infty} \left(1 + \frac{r}{n}\right)^{nt}$. Let's take the natural logarithm of both sides: $$\ln(L) = \lim_{n \to \infty} \ln\left[\left(1 + \frac{r}{n}\right)^{nt}\right] = \lim_{n \to \infty} nt \ln\left(1 + \frac{r}{n}\right)$$
  2. Reframe into $\frac{0}{0}$ Indeterminate Form: Let $x = 1/n$. As $n \to \infty$, $x \to 0$. Substituting yields: $$\ln(L) = \lim_{x \to 0} \frac{t \ln(1 + rx)}{x}$$ Plugging in $x = 0$ yields $\frac{t \ln(1)}{0} = \frac{0}{0}$, which is indeterminate.
  3. Apply L'Hôpital's Rule: Differentiate the numerator and denominator with respect to $x$: $$\text{Numerator Derivative: } \frac{d}{dx}[t \ln(1+rx)] = \frac{t \cdot r}{1+rx}$$ $$\text{Denominator Derivative: } \frac{d}{dx}[x] = 1$$
  4. Evaluate the Limit: $$\ln(L) = \lim_{x \to 0} \frac{\frac{tr}{1+rx}}{1} = tr \implies L = e^{rt}$$ Multiplying by principal $P$ yields the continuous compounding formula: $P e^{rt}$. $\blacksquare$

Exercise 3: Continuity Check of Piecewise Barrier Payoffs

An asset cashflow is modeled via a piecewise function representing a down-and-out cash payoff with barrier level $H = 80$ and digital payoff $K = 50$: $$f(S) = \begin{cases} 50 & \text{if } S \ge 80 \\ 0 & \text{if } S < 80 \end{cases}$$ Analyze the continuity of $f(S)$ at the barrier shock boundary $S = 80$ using left/right limits.

Continuity Diagnostics:
  • Right-hand limit (Above Barrier): $\lim_{S \to 80^+} f(S) = \lim_{S \to 80^+} 50 = 50$.
  • Left-hand limit (Below Barrier/Out): $\lim_{S \to 80^-} f(S) = \lim_{S \to 80^-} 0 = 0$.
  • Synthesis: Since $\lim_{S \to 80^-} f(S) \ne \lim_{S \to 80^+} f(S)$, the function has a jump discontinuity of magnitude 50 at $S=80$. This singularity requires special smoothing in numerical Finite Difference schemes to avoid instability!
