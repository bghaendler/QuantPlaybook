1.6 Random Variables

Mathematical notation, mapping sample spaces, and discrete vs continuous classifications.

1.6.1 Mathematical Notation & Mapping

Outcomes of experiments are not always numbers (e.g., getting "heads", or picking a card). A Random Variable (RV) $X$ acts as a formal function mapping elements from the sample space $\Omega$ onto the real number line $\mathbb{R}$:

$$X : \Omega \to \mathbb{R}$$

We denote random variables using uppercase letters ($X, Y$) and their observed numerical realizations using lowercase letters ($x, y$).

Two Classes of RVs:

  1. Discrete RVs: Countable number of possible outcomes (e.g. number of heads, default events, discrete asset counts).
  2. Continuous RVs: Infinite, uncountable spectrum of possible outcomes (e.g. exact portfolio return percentage, asset price drift, interest rate shifts).
