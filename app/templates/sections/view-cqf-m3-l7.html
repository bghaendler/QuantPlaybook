L7: Exotic Options

Taxonomy of Non-Vanilla Contracts, Path-Dependent PDEs, Similarity Reductions, and Monte Carlo Pricing

Classifying the Exotics Zoo

Dr. Riaz Ahmad explains that exotic options represent a major breakthrough in financial engineering, allowing virtually any bespoke terms to be embedded into a contract. Because of their customized nature and legal requirements, most exotic options are traded over the counter (OTC).

1. Time Dependence

Contracts with important dates where structural elements change (time inhomogeneous). Examples include Bermudan options (exercise only on specific dates) and resetting barriers. Time-dependence can cause sharp jumps in option values and Greeks.

2. Cash Flows

Payments made by the contract itself. Can be discrete (fixed jump $Q$ in option value on payment dates) or continuous ($Q(S)$). Discrete cash flows break analytical Black-Scholes formulas, requiring finite difference schemes.

3. Path Dependence

Payoffs affected by the entire history of the asset path. Weak path dependence (e.g. barriers) preserves the original 2D PDE but alters boundary conditions. Strong path dependence (e.g. Asian averages) introduces a new state variable, increasing the PDE's dimensionality.

4. Dimensionality

The number of random sources or state variables. Up to four dimensions, finite difference schemes and Monte Carlo are comparable. Beyond four dimensions, PDEs become computationally impractical, and Monte Carlo simulation is strongly preferred.

5. Order (Options on Options)

First-order options depend on the asset. Second-order or higher options depend on the value of *another* option. The primary example is compound options, which exist in four types: Call on Call (CoCa), Call on Put (CoP), Put on Call (PoC), and Put on Put (PoP).

Strong Path Dependence SDE & PDE Formulation

For continuously sampled path-dependent options, we track historical path information by defining a running integral state variable $I(t)$ driven by a function of the asset $f(S, \tau)$: $$I(t) = \int_{0}^{t} f(S, \tau) d\tau$$ Taking the infinitesimal change yields an ordinary differential equation in time with no diffusion term: $$dI = f(S, t) dt$$

Deriving the Path-Dependent PDE

Applying Itô's Lemma to the 3-variable option price $V(S, I, t)$ with the SDE $dS = \mu S dt + \sigma S dW$: $$dV = \frac{\partial V}{\partial S} dS + \frac{\partial V}{\partial I} dI + \left( \frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} \right) dt$$ Substitute $dI = f(S, t) dt$: $$dV = \frac{\partial V}{\partial S} dS + \left( \frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + f(S, t) \frac{\partial V}{\partial I} \right) dt$$ We set up a risk-free portfolio $\Pi = V - \Delta S$, choosing $\Delta = \frac{\partial V}{\partial S}$ to eliminate the stochastic risk $dS$. Equating the risk-free portfolio growth $d\Pi = r(V - S\frac{\partial V}{\partial S}) dt$ with the deterministic option changes yields the Strongly Path-Dependent PDE: $$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + f(S, t) \frac{\partial V}{\partial I} + rS \frac{\partial V}{\partial S} - rV = 0$$

Dimensionality Reduction via Similarity Transform

For an arithmetic average Asian option, $f(S, t) = S$, so $I(t) = \int_0^t S(\tau)d\tau$. The pricing PDE has 3 variables ($S, I, t$). By exploiting the mathematical structure of the payoff (for a floating strike call: $\max(S - I/T, 0)$), we can perform a similarity reduction to reduce the PDE to 2 variables ($R, t$), where: $$R = \frac{I}{S}$$ We propose the ansatz $V(S, I, t) = S W(R, t)$.

Reduced PDE Derivation Steps

Using the chain rule, we compute the partial derivatives: $$\frac{\partial V}{\partial t} = S \frac{\partial W}{\partial t}, \quad \frac{\partial V}{\partial I} = \frac{\partial W}{\partial R}, \quad \frac{\partial V}{\partial S} = W - R \frac{\partial W}{\partial R}, \quad \frac{\partial^2 V}{\partial S^2} = \frac{R^2}{S} \frac{\partial^2 W}{\partial R^2}$$ Substituting these into the strongly path-dependent Asian PDE: $$S \frac{\partial W}{\partial t} + \frac{1}{2}\sigma^2 S^2 \left( \frac{R^2}{S} \frac{\partial^2 W}{\partial R^2} \right) + S \frac{\partial W}{\partial R} + rS \left( W - R \frac{\partial W}{\partial R} \right) - r S W = 0$$ Dividing by the stock price $S$ simplifies to the Reduced Asian PDE: $$\frac{\partial W}{\partial t} + \frac{1}{2}\sigma^2 R^2 \frac{\partial^2 W}{\partial R^2} + (1 - rR) \frac{\partial W}{\partial R} = 0$$

Discrete Updating Rules

In the real world, option paths are sampled at discrete dates $t_1, t_2, \dots, t_M$ rather than continuously. To prevent having to save the entire historical path (which is highly inefficient), we use updating rules that express the state variable dynamically using only the most recent sample.

1. Discrete Arithmetic Average

The arithmetic average $A_M$ of $M$ samples: $$A_M = \frac{1}{M}\sum_{k=1}^M S(t_k)$$ At sampling date $t_i$, the new average $A_i$ is computed from the previous average $A_{i-1}$ and today's stock price $S(t_i)$: $$A_i = \frac{1}{i} S(t_i) + \frac{i-1}{i} A_{i-1}$$

2. Discrete Lookback Maximum

For lookbacks (options whose payoffs depend on the absolute peak/valley of the path), the realized maximum $I_i$ is updated on sampling date $t_i$ according to: $$I_i = \max\left(S(t_i), I_{i-1}\right)$$ Generalizing updating rules for any path-dependent quantity: $$I_i = F\left(S(t_i), I_{i-1}, i\right)$$

Exercise 1: Power Option with Continuous Cash Flow
Problem:
An option pays a continuous cash flow rate $S^2 dt$ during interval $dt$. At expiry, the payoff is $V(S, T) = S^2$. Derive the non-homogeneous pricing PDE and solve for $\phi(t)$ assuming a solution $V(S, t) = \phi(t)S^2$.
Detailed Solution:

1. Non-Homogeneous PDE Derivation:
Portfolio $\Pi = V - \Delta S$. Infinitesimal change includes the continuous cash flow: $$d\Pi = dV - \Delta dS + S^2 dt$$ Apply Itô's Lemma to expand $dV$, and choose $\Delta = \frac{\partial V}{\partial S}$ to eliminate the $dS$ risk: $$d\Pi = \left( \frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + S^2 \right) dt$$ Under no-arbitrage, $d\Pi = r\Pi dt = r(V - S\frac{\partial V}{\partial S})dt$. Equating and dividing by $dt$: $$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + rS\frac{\partial V}{\partial S} - rV = -S^2$$

2. Solving the ODE for $\phi(t)$:
Substitute $V = \phi(t)S^2$ and its derivatives into the PDE, dividing through by $S^2$: $$\dot{\phi}(t) + \sigma^2 \phi(t) + 2r\phi(t) - r\phi(t) = -1 \implies \dot{\phi}(t) + (\sigma^2 + r)\phi(t) = -1$$ Using integrating factor $e^{(\sigma^2+r)t}$, integrating from $t$ to $T$, and using the final boundary condition $\phi(T) = 1$ yields: $$\phi(t) = \frac{1}{\sigma^2 + r} \left( \left(\sigma^2 + r + 1\right) e^{(\sigma^2 + r)(T-t)} - 1 \right)$$

Exercise 2: Cauchy-Euler Separable Solutions
Problem:
Solve the Black-Scholes separable equation $V(S, t) = f(S)g(t)$ for Cauchy-Euler roots: $$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + (r-D)S\frac{\partial V}{\partial S} - rV = 0$$
Detailed Solution:

1. Splitting into ODEs:
Substitute $V(S,t) = f(S)g(t)$ and divide by $fg$ to isolate variables equal to constant $\lambda$: $$\frac{dg}{dt} - \lambda g = 0 \implies g(t) = e^{\lambda t}$$ $$\frac{1}{2}\sigma^2 S^2 f'' + (r-D)Sf' + (\lambda - r)f = 0$$

2. Solving Cauchy-Euler $f(S) = S^\alpha$:
Substitute $f(S) = S^\alpha$ to obtain the quadratic auxiliary equation: $$\alpha^2 + \left( \frac{2(r-D)}{\sigma^2} - 1 \right)\alpha + \frac{2(\lambda - r)}{\sigma^2} = 0$$ Solving the quadratic roots $\alpha_\pm = \left(\frac{1}{2} - \frac{r-D}{\sigma^2}\right) \pm \bar{d}_\pm$ where: $$\bar{d}_\pm = \pm \sqrt{ \left( \frac{r-D}{\sigma^2} - \frac{1}{2} \right)^2 - \frac{2(\lambda - r)}{\sigma^2} }$$ - Distinct Roots: $V(S, t) = e^{\lambda t} S^{\frac{1}{2} - \frac{r-D}{\sigma^2}} \left( A S^{\bar{d}_+} + B S^{\bar{d}_-} \right)$ - Repeated Root: Occurs when $\lambda = r + \frac{\sigma^2}{2} \left( \frac{r-D}{\sigma^2} - \frac{1}{2} \right)^2$, yielding: $$V(S, t) = e^{\lambda t} S^{\frac{1}{2} - \frac{r-D}{\sigma^2}} \left( \epsilon + \zeta \log S \right)$$

Exercise 3: Pricing with Dividends & Cash Flows
Problem:
For an asset paying dividend yield $D S dt$, and an option paying continuous cash flow $f(t) S dt$, show the derivative's delta is: $$\Delta(S, t) = e^{-D(T-t)} + \int_{t}^{T} e^{-D(\tau - t)} f(\tau) d\tau$$
Detailed Solution:

1. Portfolio SDE with Dividends:
Since the portfolio is short the stock, we must pay the dividend yield $D\Delta S dt$. Holding the option pays $f(t)S dt$: $$d\Pi = dV - \Delta dS - D \Delta S dt + f(t)S dt$$ Choosing $\Delta = \frac{\partial V}{\partial S}$ and equating to $r\Pi dt$ yields: $$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + (r-D)S\frac{\partial V}{\partial S} - rV = -f(t)S$$

2. Solving for Delta:
Substitute $V(S,t) = \phi(t)S$, which simplifies to: $$\dot{\phi}(t)S + (r-D)S\phi(t) - rS\phi(t) = -f(t)S \implies \dot{\phi}(t) - D\phi(t) = -f(t)$$ Using integrating factor $e^{-Dt}$ and terminal boundary condition $\phi(T) = 1$: $$\phi(t) = e^{-D(T-t)} + \int_{t}^{T} e^{-D(\tau - t)} f(\tau) d\tau$$ Since $\Delta(S,t) = \frac{\partial V}{\partial S} = \phi(t)$, we obtain the required delta.
