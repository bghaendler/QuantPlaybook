Second Order Ordinary Differential Equations - Special Cases & Operator Theory

Decomposition of special cases, operator notation, linear independence, vector spaces, and complementary/particular splits.

1. General Form & Decomposition of Special Cases

The most general form of a second-order ODE can be written implicitly as $F(x, y, y', y'') = 0$. When one or more terms are explicitly absent, we can decompose the equation into a sequential system of first-order equations:

Case A: Both $y'$ and $y$ are missing: $y'' = f(x)$

Solved by direct double integration. For example, if $y'' = 4x$:

$$y' = 2x^2 + C \implies y = \frac{2}{3}x^3 + Cx + D$$

Every second-order ODE solution must contain exactly two arbitrary constants of integration.

Case B: Variable $y$ is missing: $y'' = f(x, y')$

Introduce substitution $p = y'$, which means $y'' = p'$. This converts the second-order equation into a first-order equation in $p$:

$$\frac{dp}{dx} = f(x, p)$$

Once $p(x)$ is solved, recover $y$ via integration: $y = \int p(x)\,dx$.

Case C: Independent variable $x$ is missing: $y'' = f(y)$

Using substitution $p = y'$, because $x$ is missing, we use the chain rule to rewrite the second derivative:

$$y'' = \frac{dp}{dx} = \frac{dp}{dy}\frac{dy}{dx} = p \frac{dp}{dy}$$

This transforms the ODE into a first-order separable equation in terms of variables $p$ and $y$:

$$p \frac{dp}{dy} = f(y)$$

2. Nth Order Linear Equations & Operator Notation

Using the differential operator notation where $D \equiv d/dx$, we can express an $n^{\text{th}}$-order linear differential equation compactly as:

$$Ly = g(x)$$

where $L$ is a linear differential operator. The linearity of $L$ is defined by two fundamental algebraic properties:

  • Constants factor out: $L(c y) = c L(y)$
  • Linearity over sums: $L(y_1 + y_2) = L(y_1) + L(y_2)$

The general solution to a non-homogeneous equation $Ly = g(x)$ is the sum of two distinct components:

$$y = y_c + y_p$$

Where $y_c$ (Complementary Function) solves the homogeneous system $Ly = 0$, and $y_p$ (Particular Integral) solves the non-homogeneous equation $Ly = g(x)$.

3. Homogeneous Solution & Vector Space Properties

Because the operator $L$ is linear, the set of all solutions to the homogeneous equation $Ly = 0$ forms a vector space. This means that if $y_1$ and $y_2$ are solutions, any linear combination $c_1 y_1 + c_2 y_2$ is also a solution.

  • A set of functions is linearly dependent if a linear combination of them can equal zero with not all coefficients being zero, meaning one function can be expressed in terms of the others.
  • A set of functions is linearly independent if the only way the linear combination can equal zero is for all coefficients to be zero.

An $n^{\text{th}}$-order homogeneous ODE has exactly $n$ linearly independent solutions. The general solution of the homogeneous equation is formed by taking a linear combination of all $n$ independent solutions:

$$y_c = c_1 y_1 + c_2 y_2 + \dots + c_n y_n$$

We substitute $p = y'$ to reduce it to a first-order linear ODE: $x p' + 2p = x^3 \implies p' + \frac{2}{x} p = x^2$.
