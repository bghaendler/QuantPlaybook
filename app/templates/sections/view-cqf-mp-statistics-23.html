2.3 Regression & Correlation

Method of least squares, regression line partial derivatives, and Pearson/Spearman coefficient calculations.

2.3.1 Ordinary Least Squares Regression Derivation

A sensible first step to identify relationships between variables is to plot the bivariate data on a scatter diagram, i.e. pairs of values $(x_i, y_i)$.

We want to fit a straight line $y = bx + a$ through the data so as to minimise the sum of the squared distances of the points from the line. The difference between the data value and the fitted line is called the residual or error.

The residual of the $i^{\text{th}}$ data point $(x_i, y_i)$ is:

$$r_i = y_i - y = y_i - (b x_i + a)$$

The sum of squared residuals ($S.R.$) is defined as:

$$S.R. = \sum_{i=1}^{n} r_i^2 = \sum_{i=1}^{n} [y_i - (b x_i + a)]^2$$

Step 1: Expand the Sum of Squared Residuals $$S.R. = \sum \left[ y_i^2 - 2y_i(bx_i + a) + (bx_i + a)^2 \right]$$ $$= \sum \left[ y_i^2 - 2by_ix_i - 2ay_i + b^2 x_i^2 + 2bax_i + a^2 \right]$$ $$= n\bar{y}^2 - 2bn\bar{xy} - 2an\bar{y} + b^2 n\bar{x}^2 + 2ban\bar{x} + na^2$$ Step 2: Take Partial Derivatives to Minimize

To find the coordinates that minimize the error, we set partial derivatives to zero:

$$\frac{\partial(S.R.)}{\partial b} = -2n\bar{xy} + 2bn\bar{x}^2 + 2an\bar{x} = 0$$ $$\frac{\partial(S.R.)}{\partial a} = -2n\bar{y} + 2bn\bar{x} + 2an = 0$$ Step 3: Solve the Linear Simultaneous Equations

$$b = \frac{S_{xy}}{S_{xx}}$$

$$a = \bar{y} - b\bar{x}$$

Where: $$S_{xx} = \sum (x_i - \bar{x})^2 = \sum (x_i^2) - \frac{(\sum x_i)^2}{n}$$ $$S_{xy} = \sum (x_i - \bar{x})(y_i - \bar{y}) = \sum x_i y_i - \frac{(\sum x_i)(\sum y_i)}{n}$$

2.3.2 Pearson Product-Moment Correlation Coefficient

A measure often used within statistics to quantify linear dependence is the Pearson product-moment correlation coefficient (PMCC), denoted by $r$. This coefficient gives a value between +1 and -1:

$$\text{PMCC } r = \frac{S_{xy}}{\sqrt{S_{xx}S_{yy}}}$$

Where: $$S_{yy} = \sum (y_i - \bar{y})^2 = \sum (y_i^2) - \frac{(\sum y_i)^2}{n}$$

Worked Example:

Data Table:

$x$ 510152025303540
$y$ 9890816661473934

Summation metrics calculated: $\sum x_i = 180, \sum y_i = 516, \sum x_i^2 = 5100, \sum y_i^2 = 37228, \sum x_i y_i = 9585$

$$S_{xy} = 9585 - \frac{180 \times 516}{8} = -2025$$

$$S_{xx} = 5100 - \frac{180^2}{8} = 1050 \implies b = \frac{-2025}{1050} = -1.929$$

$$a = \bar{y} - b\bar{x} = 64.5 - (-1.929 \times 22.5) = 107.9$$

OLS regression line equation: $y = -1.929x + 107.9$

PMCC Calculation: $S_{yy} = 37228 - \frac{516^2}{8} = 3946$

$$r = \frac{-2025}{\sqrt{1050 \times 3946}} = -0.995 \quad \text{(Strong negative correlation)}$$

2.3.3 Spearman's Rank Correlation Coefficient

Instead of dealing with absolute values, we can assign a rank to each value and calculate a correlation based on these ranks, serving as an approximation to PMCC:

$$r_s = 1 - \frac{6 \sum d_i^2}{n(n^2 - 1)}$$

Where $d_i$ is the difference in ranks for each pair, and $n$ is the number of pairs.

Worked Judge Example:

Judge X31675482
Judge Y21584376
Diff $d$101-1111-4
$d^2$101111116

Sum of $d^2 = 22$, sample size $n = 8$:

$$r_s = 1 - \frac{6 \times 22}{8(8^2 - 1)} = 0.738$$
