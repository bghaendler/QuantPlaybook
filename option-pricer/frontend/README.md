# Option Pricer - Frontend

This is the React frontend for the Option Pricer application.

## Getting Started

For complete installation and running instructions on macOS, please see the **[main README](../README.md)** in the project root.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Technology Stack

- **React 19.2.0** - UI framework
- **Vite (Rolldown)** - Build tool and dev server
- **Plotly.js** - Interactive charts and visualizations
- **Axios** - HTTP client for API calls
- **Recharts** - Additional charting library
- **KaTeX** - Mathematical formula rendering
- **Lucide React** - Icon library

## Requirements

- Node.js 16 or higher
- npm or yarn

For full documentation, troubleshooting, and backend setup, see the [main README](../README.md).
